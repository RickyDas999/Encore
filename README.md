# Encore

Encore is a full-stack event discovery and trip planning app. Users can browse
events, save favorites, and create simple trip plans around concerts, sports,
comedy shows, and other live experiences in nearby cities.

## Project Structure

- `client`: React + TypeScript + Vite frontend
- `server`: Node.js + TypeScript + Express backend

## Next Build Steps

1. Add PostgreSQL connection and schema files in `server/src/db`.
2. Seed one user and a set of sample events.
